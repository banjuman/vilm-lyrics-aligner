# Contributing

Thanks for helping improve Vilm Lyrics Aligner.

Before opening a pull request:

1. Keep the supplied lyrics as the sole subtitle text source.
2. Add focused tests for behavioral changes and run the full Python suite.
3. Build the Avalonia desktop project when changing desktop code.
4. Keep English and Korean UI copy synchronized.
5. Do not commit copyrighted lyrics, client media, diagnostic output containing
   user content, downloaded models, private runtimes, or generated installers.

Timing changes should be supported by repeatable examples from more than one
song or speaker. A heuristic that fixes one recording while weakening general
behavior is unlikely to be accepted.

Use GitHub Issues for reproducible bugs and feature discussions. Do not include
private media or credentials in an issue.
